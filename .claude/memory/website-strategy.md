## 2. SEO & LLM OPTIMIZATION STRATEGY

### The Paradigm Shift

**Traditional SEO:** Keyword-based, Google-centric
**New Reality:** Conversational AI search (ChatGPT, Perplexity, Claude, Gemini)

**Critical Stat:** 52% of sources cited in AI-generated answers come from Google's first page
**Implication:** You STILL need traditional SEO, but LLM optimization is now equally important

### LLM Referral Traffic Value

**Case Study: Logikcull**
- June 2023: 5% of ALL leads came from ChatGPT
- Revenue: $100,000 MRR from ChatGPT referrals alone
- Conversion: LLM traffic converts HIGHER than organic search
- Engagement: Longer time on site

**Why LLM Traffic is Better:**
- Users asking specific, qualified questions
- Already researched alternatives (decision stage)
- Higher intent = higher conversion

### Generative Engine Optimization (GEO) Checklist

#### 1. Semantic HTML (LLM-Friendly Structure)
✅ Clean, valid HTML5
✅ Proper heading hierarchy (H1 → H2 → H3)
✅ Descriptive alt text on ALL images
✅ Semantic tags (`<article>`, `<section>`, `<nav>`)
✅ ARIA labels where appropriate

#### 2. Schema Markup (Structured Data)
**Priority schemas to add:**
- ✅ `Person` - For Emanuel Flury
- ✅ `Organization` - For eflury.com & Taxed GmbH
- ⚠️ `FAQPage` - 20-30 questions (CRITICAL for LLMs)
- ⚠️ `Article` - For blog posts
- ⚠️ `HowTo` - For implementation guides
- ⚠️ `Service` - For each package

**Example FAQ Schema (LLMs LOVE this):**
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "Why choose Claude over ChatGPT for Swiss SME automation?",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "Claude excels at..."
    }
  }]
}
```
